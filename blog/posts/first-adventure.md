## 冒险背景

作为一名初出茅庐的前端冒险者，你站在迷雾森林的边缘。这里的一切都充满神秘——会说话的狐狸（浏览器）、发光的蘑菇（HTML标签）、飘浮的符文（CSS样式）。今天，你将踏出成为前端法师的第一步。

## 第一章：准备你的冒险装备（开发环境搭建）

### 1.1 选择你的法杖（代码编辑器）

在迷雾森林中，你需要一把称手的法杖来书写魔法咒语（代码）：

| 法杖名称 | 特点 | 适合人群 |
|---------|------|---------|
| VS Code | 最强大的法杖，拥有无数魔法插件 | 所有冒险者 |
| Sublime Text | 轻量快速，启动如闪电 | 追求速度的老手 |
| WebStorm | 功能全面，但需要更多魔法能量 | 专业法师 |

**我的推荐：** 新手请选择 VS Code，并安装以下魔法插件：
- Live Server：让你实时看到咒语效果
- Prettier：自动美化你的魔法符文
- Auto Rename Tag：自动闭合你的魔法标记

### 1.2 召唤第一个浏览器（Chrome开发者工具）

按下 F12 键，你将召唤出浏览器的秘密神器——开发者工具。这里可以：
- 查看任何网页的魔法结构（Elements面板）
- 调试你的咒语错误（Console面板）
- 查看网络请求的魔法波动（Network面板）

## 第二章：认识迷雾森林的生物（HTML基础）

### 2.1 会说话的狐狸：HTML是什么？

HTML（HyperText Markup Language）就像是迷雾森林的语言，它告诉浏览器：
- 哪里是标题（会说话的狐狸）
- 哪里是段落（发光的蘑菇）
- 哪里是图片（会飞的树叶）

### 2.2 第一次召唤：创建你的第一个HTML文档

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>迷雾森林的初探</title>
</head>
<body>
    <!-- 这是注释，只有冒险者能看到 -->
    <h1>欢迎来到迷雾森林</h1>
    <p>我是会说话的狐狸，很高兴见到你！</p>
    <img src="magic-mushroom.jpg" alt="发光的蘑菇">
</body>
</html>
```

### 2.3 魔法标记详解（常用HTML标签）

| 标签名 | 中文名 | 作用 | 就像... |
|--------|--------|------|---------|
| `<h1>` ~ `<h6>` | 标题 | 定义各级标题 | 森林里的路牌 |
| `<p>` | 段落 | 包裹一段文字 | 狐狸说的话 |
| `<img>` | 图片 | 显示图片 | 发光的蘑菇 |
| `<a>` | 链接 | 跳转到其他页面 | 传送门 |
| `<div>` | 容器 | 装其他元素的盒子 | 魔法背包 |
| `<span>` | 小容器 | 装一小段文字 | 魔法口袋 |

## 常见问题（QA）

**Q：为什么我的代码在浏览器里显示不出来？**
A：检查你的文件名是否以 `.html` 结尾，并且用浏览器打开了正确的文件。

**Q：VS Code 怎么创建新文件？**
A：快捷键 Ctrl+N（Windows）或 Cmd+N（Mac），保存时记得加 `.html` 后缀。

**Q：图片显示不出来怎么办？**
A：检查图片路径是否正确，或者先用网络图片试试（比如 `https://picsum.photos/200/300`）。

# 迷雾森林的初次冒险

作为一名初出茅庐的冒险者，我在迷雾森林遇到了会说话的狐狸和发光的蘑菇...

---

晨雾像薄纱一样缠绕在古老的橡树之间。我握紧新手村里铁匠铺买来的短剑——说实话，它可能连成熟的南瓜都劈不开——踏入了迷雾森林。

第一个小时，我迷路了三次。第二个小时，我开始怀疑指南针是不是被森林里的什么魔法影响了。

然后我遇到了它。

一只银灰色的狐狸坐在一块长满青苔的石头上，歪着头看我。它的眼睛是琥珀色的，在雾气中微微发光。

"新人？"它开口说。

我差点把短剑扔出去。

狐狸似乎对我的反应很满意，摇了摇尾巴："往北走有个发光的蘑菇圈，别踩进去。往东走有条小溪，水里的石头可以当火石用。往西——算了，往西是悬崖。"

"你为什么...会说话？"

狐狸叹了口气，那种"年轻人问题真多"的叹气："在这片森林里活了三百多年，不会说话才奇怪吧？"

它跳下石头，示意我跟上。我们穿过雾气弥漫的小径，绕过几棵看起来正在打盹的树人，最后停在一片空地上。

蘑菇们正在发光。

不是比喻，是真的发光。蓝色的、紫色的、偶尔还有橙色的光从菌盖上流淌下来，照亮了周围的雾气，像一个微型的水下世界。

"月光菌，"狐狸说，"只在雾天发光。摘下它们，三个小时后光就会消失。可以用来做药剂，也可以在黑暗的地方当灯。"

它看了我一眼："不过记住，采蘑菇前要和它们说声谢谢。这是规矩。"

我小心翼翼地蹲下身，对着一簇蓝色的蘑菇轻声说："谢谢。"

蘑菇们似乎闪得更亮了一些。

接下来的三天，我在迷雾森林里学会了很多东西：哪些树的汁液可以治伤口，如何分辨幽灵蘑菇和可食用蘑菇的区别（幽灵蘑菇会让你看到已经去世的奶奶，虽然很温馨，但很危险），以及最重要的一课——

那个傍晚，狐狸带我来到森林深处的一块空地。月亮刚从雾气中露出脸，月光洒下来，整个空地亮如白昼。

"看，"狐狸说。

空地的中央，成千上万的发光蘑菇同时闪烁起来，像一场无声的烟火。光点升起，在雾气中漂浮，和月光混在一起，分不清哪是光，哪是雾。

"这是月光菌的庆典，"狐狸轻声说，"每个月圆之夜都会举行。它们邀请所有森林里的生灵来看。"

我站在那里，忘记了时间，忘记了迷路，忘记了我只是一个拿着破短剑的菜鸟冒险者。

回程的时候，我问狐狸："以后我还能来找你吗？"

狐狸的尾巴在雾气中摇了摇："这片森林又不会跑掉。不过下次来，带点蜂蜜。我爱吃甜的。"

然后它消失在雾气里，留下我一个人站在森林边缘。

晨光正在驱散薄雾，村庄的轮廓在远方若隐若现。我低头看手里的月光菌，它还在发着淡淡的蓝光。

狐狸的声音不知道从哪里飘来："对了，忘了自我介绍。我叫星语者。如果你迷路了，就在心里喊我的名字。"

我回头望去，雾气深处，似乎有一双琥珀色的眼睛在看着我。

然后雾气合拢，森林恢复了它千百年来的宁静。

---

**后记**：月光菌的发光效果会在三个时辰后消失，但如果泡在蜂蜜水里，可以延长到五个时辰。这个发现让我在冒险者公会换到了第一笔赏金——五个银币和一罐蜂蜜。

现在我终于知道下次带什么礼物去见星语者了。
